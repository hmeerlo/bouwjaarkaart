Embed Demo

Bouwjaarkaart op jouw website

Voeg een interactieve kaart met bouwjaren van alle gebouwen in Nederland toe aan je blog, artikel of website. Gratis, twee regels code.

Live voorbeeld

De grachtengordel van Amsterdam. Scroll en klik op gebouwen om het bouwjaar te zien.

Embed code
<!-- Plak dit waar je de kaart wilt tonen --> <div data-stadsprint-bouwjaarkaart data-lat="52.3700" data-lon="4.8890" data-zoom="15" data-height="450"></div> <!-- Voeg dit eenmalig toe, onderaan je pagina --> <script src="https://stadsprint.nl/embed/bouwjaarkaart.js" async></script>

Meer steden

Parameters

Pas de kaart aan met deze data-attributen.

Attribuut Beschrijving Default
data-lat Breedtegraad (latitude) 52.37
data-lon Lengtegraad (longitude) 4.90
data-zoom Zoomniveau (12 t/m 18) 14
data-height Hoogte in pixels 400
data-legend Legenda tonen (true / false) true
